# joshmolnar.com

Personal brand + entity site for **Josh Molnar** (founder of Bitcoin Daily). Static HTML/CSS,
no build step.

## Deploy
Hosted on **Netlify**, auto-deploys on every push to `main`.
- Build command: _(none)_
- Publish directory: repo root (`.`)

Connected custom domains: `joshmolnar.com` (primary), with `joshuamolnar.com` and `bitcoinjae.com`
301-redirecting to it.

## Edit
Edit the `.html` / `styles.css` files directly and push — Netlify rebuilds automatically.
After adding a new blog post, also add its URL to `sitemap.xml`.
